# 如何手动启动网站服务

如果您发现网站无法访问（例如电脑重启后），请按照以下步骤手动启动：

1. **双击 `start_services.bat`**：
   - 在项目根目录下找到 `start_services.bat` 文件并双击运行。
   - 这将打开两个命令行窗口：一个运行后端服务，另一个运行前端服务。

2. **等待启动**：
   - 脚本会自动等待几秒钟，确保服务启动成功。
   - 随后会自动在您的默认浏览器中打开 `http://localhost:5173`。

3. **保持窗口开启**：
   - 在使用网站期间，请**不要关闭**新打开的两个命令行窗口。如果关闭了，服务也会停止。

---

### 故障排除
- 如果浏览器打开后显示“无法连接”，请检查命令行窗口中是否有报错信息。
- 确保您的电脑上已安装 Node.js 和 Python 环境。
